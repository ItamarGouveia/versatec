# Bem-vindo ao repositório iPrescrição!

iPrescrição é um projeto de software web que tem como objetivo fornecer uma plataforma fácil e segura para o cadastro de prescrição médica. Com o crescente número de pacientes e a necessidade de prescrição de medicamentos precisos, é essencial que os profissionais de saúde tenham acesso a uma ferramenta eficiente e confiável que lhes permita prescrever medicamentos com facilidade e precisão.

### 🔧 Instalação


Para baixar os fontes é só seguir esses passos:

```
git clone https://github.com/ItamarGouveia/versatec.git
```

```
npm install
```

```
npm start
```

Lembrando que você deve configurar o firebase antes desses passos
